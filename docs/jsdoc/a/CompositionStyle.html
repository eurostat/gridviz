Class: CompositionStyle

CompositionStyle(opts)

A style showing the composition of a total in different categories, with different color hues. It consists of a symbol with different parts, whose size reflect the proportion of the corresponding category. For a list of supported symbols, @see CompositionType The symbol can be scaled depending on the cell importance.
